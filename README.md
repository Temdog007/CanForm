# CanForm (Canvas Form)

Create forms that appear as dialog windows to edit data

Supports
| Data Type 	 | Widget     |
| -------------- | ---------- |
| Boolean   	 | Checkbox   |
| String    	 | Textbox    |
| Set of Strings | Combox Box |

## Backends
- [x] Implemented with GTK for desktops
- [x] Implemented with Emscripten for web browsers
- [ ] Implemented for Android
- [ ] Implemented for iOS
